# song-emotion-model
Making a model to predict the emotional effect of songs.


Note that in the data files, emotional states such as 'before: sad to happy' are stored as numbers from 0 to 1. With 0 being sad and 1 being happy.

While 'delta: sad to happy' would be stored as a number from -1 to 1. with -1 meaning a change from happy to sad, and 1 meaning a change from sad to happy.
